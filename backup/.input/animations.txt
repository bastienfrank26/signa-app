Ajoute des animations premium à la landing page Signa.

Stack :
React + TypeScript + Vite + Tailwind.

Utilise Framer Motion.

Installer si nécessaire :
npm install framer-motion

Objectif :
Ajouter des animations subtiles, professionnelles et fluides sans changer le design global.

À animer :

1. Hero
- titre : fade-up
- sous-titre : fade-up avec délai léger
- boutons : fade-up avec délai
- badges : apparition progressive
- mockup visuel : léger effet flottant vertical infini

2. Sections
- chaque section doit apparaître au scroll avec fade-up
- utiliser viewport once: true
- animation courte et fluide

3. Cartes
- cartes services, réalisations et forfaits : stagger animation
- hover : légère élévation, scale 1.02, shadow/glow orange subtil

4. CTA
- bouton principal : hover glow orange
- CTA final : apparition plus marquée au scroll

5. Formulaire EvaluationForm
- transition fluide entre les étapes
- utiliser AnimatePresence
- fade/slide horizontal léger
- barre de progression animée

Contraintes :
- Ne pas modifier les textes
- Ne pas changer les couleurs
- Ne pas changer la structure globale
- Ne pas alourdir visuellement le site
- Respecter prefers-reduced-motion si possible
- Les animations doivent rester premium, rapides et discrètes

Créer au besoin :
src/lib/motion.ts

Y placer des variants réutilisables :
- fadeUp
- staggerContainer
- cardHover
- slideStep

Appliquer les variants aux composants existants :
- Hero.tsx
- BeforeAfter.tsx
- WhySigna.tsx
- Realisations.tsx
- Pricing.tsx
- EvaluationForm.tsx
- CtaBand.tsx

Après modification :
- lancer npm run build
- corriger toute erreur TypeScript
